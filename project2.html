Text-to-Speech System with Hinglish Accent

Description: This project was aimed at developing a Text-to-Speech (TTS) system with a specific focus on generating speech in the Indore accent. The goal was to create a natural-sounding voice output that resonates with the regional dialect and pronunciation style of Indore. While the project successfully implemented a Hinglish-based TTS system, achieving the nuanced Indore accent proved challenging due to certain technical limitations.

Key Features:

Hinglish Text-to-Speech Conversion: The system effectively converts text input into Hinglish phonetics, enabling smooth and natural-sounding conversational flow.

Language Processing: The system integrates a language processing layer that interprets and converts various languages (like Hindi and English) into a Hinglish representation.

Voice SynthesisUtilized TTS libraries like Google Text-to-Speech (gTTS) and pyttsx3 for generating realistic voice outputs.

Challenges with Indore Accent Implementation:

Accent Data Availability:The lack of a publicly available dataset specific to the Indore accent made it challenging to train a TTS model accurately.

Dialectical Variations: Capturing the unique pronunciations, rhythm, and intonations of the Indore accent required advanced speech synthesis techniques and a dataset tailored for linguistic fine-tuning.

Pre-built Limitations: Most TTS engines, such as Google TTS or AWS Polly, provide generalized accents (e.g., Indian English or Hindi), which do not support region-specific customizations like the Indore accent.

Resource Constraints Developing a custom TTS model for the Indore accent would require significant computational resources, advanced phonetic mapping, and proprietary training datasets.

Technologies Used:

Python Libraries: Used gTTS, pyttsx3, and SpeechRecognition for voice synthesis and text processing.

Phonetic Conversion: Applied custom logic for converting text into Hinglish phonetics.

Integration:Developed a standalone application with an easy-to-use interface for generating Hinglish TTS outputs.

Impact:Although the specific Indore accent could not be achieved due to data and resource limitations, the project delivered a Hinglish TTS system that provides clear, conversational voice output. This project highlights the importance of linguistic and cultural nuances in TTS systems and sets the foundation for future work on regional accent synthesis.
